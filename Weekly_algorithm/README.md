# 주간 알고리즘 학습

## 학습 목표

- 알고리즘 이론 학습과 문제 풀이를 병행해야 하는 유형들에 대해 문제를 풀어본다.

## 학습 방법

1. 문제 풀이 후 해당 코드 본인 `개인 브랜치` 이용해서 업로드

2. `documents` branch에서 README.md 수정 후 commit 남기기

- 본인이 풀어본 문제 중 인상적인 그리디 문제 업로드 가능
